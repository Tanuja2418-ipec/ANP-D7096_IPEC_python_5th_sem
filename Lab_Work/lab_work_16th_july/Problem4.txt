Problem 4: Employee Salary Calculator 
Create the following functions: 
calculate_hra(basic) 
calculate_da(basic) 
calculate_tax(gross_salary) 
calculate_net_salary(basic) 
Rules: 
• HRA = 20%  
• DA = 10%  
• Tax = 8% of Gross Salary  

Input: Basic Salary
Output: 
Basic Salary 
HRA 
DA 
Gross Salary 
Tax 
Net Salary 

#------------------------Coding Section--------------------------------


# Employee Salary Calculator

# Function to calculate HRA (20% of Basic Salary)
def calculate_hra(basic):
    return basic * 0.20


# Function to calculate DA (10% of Basic Salary)
def calculate_da(basic):
    return basic * 0.10


# Function to calculate Tax (8% of Gross Salary)
def calculate_tax(gross_salary):
    return gross_salary * 0.08


# Function to calculate Net Salary
def calculate_net_salary(basic):

    # Calculate HRA
    hra = calculate_hra(basic)

    # Calculate DA
    da = calculate_da(basic)

    # Calculate Gross Salary
    gross_salary = basic + hra + da

    # Calculate Tax
    tax = calculate_tax(gross_salary)

    # Calculate Net Salary
    net_salary = gross_salary - tax

    # Display Salary Details
 
    print("Basic Salary : Rs.", basic)
    print("HRA          : Rs.", hra)
    print("DA           : Rs.", da)
    print("Gross Salary : Rs.", gross_salary)
    print("Tax          : Rs.", tax)
    print("Net Salary   : Rs.", net_salary)


# Taking Basic Salary as input
basic_salary = float(input("Enter Basic Salary: ₹"))

# Calling the function to calculate net salary
calculate_net_salary(basic_salary)
